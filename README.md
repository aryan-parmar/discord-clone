## Getting Started

```sh 
git clone https://github.com/aary114/discord-clone.git
```

### Prerequisites

-[ Node ](https://nodejs.org/en/)

## Usage

Run
```sh
cd backend
npm i
node index.js
```
```sh
cd client
npm i
npm run
```
